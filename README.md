# Configuraci√≥n de la IP del servidor üñ•Ô∏è

Para poder ejecutar nuestro programa en ordenadores remotos, hay 2 archivos que debemos modificar: **config.server y config.client**. Esto con el fin de poner la direcci√≥n IP de la computadora que ser√° el Servidor en el atributo **Ice.Default.Host**. Si el atributo **Ice.Default.Host=localhost**, el programa se ejecutar√° localmente.

### server/src/main/resources/config.server ###

```
#
# The server creates one single object adapter with the name
# "Hello". The following line sets the endpoints for this
# adapter.
#
# Printer.Endpoints=tcp -p 9099
#
# Only listen on the ZeroTier's LIASOn1 interface by default.
# 10.147.19.157 - xhgrid1
# 10.147.19.34 - xhgrid3
# 10.147.19.26 - xhgrid4
#
#Ice.Default.Host=localhost

Service.Endpoints = default -p 8000
Ice.ThreadPool.Server.Size=3

```
